# 🐍 Python – Estruturas de Repetição com `for`

Este repositório reúne atividades desenvolvidas no **Google Colab** com foco no uso da estrutura de repetição **`for`** em Python.

Ao longo dos exercícios, foram aplicados conceitos de iteração sobre listas, processamento de dados, cálculo de bônus, contagem de resultados, identificação de valores e utilização de estruturas condicionais dentro dos laços de repetição, simulando cenários encontrados em aplicações reais.

---

## 🎯 Objetivos do Projeto

* Aprender o funcionamento da estrutura de repetição `for`.
* Percorrer listas e processar seus elementos.
* Integrar estruturas de repetição com condicionais (`if` e `else`).
* Desenvolver algoritmos para análise de dados.
* Aplicar regras de negócio em situações práticas.

---

## 📚 Conteúdo Estudado

Antes das atividades práticas, foram realizados exercícios introdutórios para compreender o funcionamento do laço `for`, incluindo:

* Repetições utilizando `range()`.
* Percorrendo listas de strings.
* Percorrendo listas numéricas.
* Processamento de vendas utilizando listas.
* Contagem de elementos que atendem a uma condição.

---

## 📚 Atividades Desenvolvidas

### 🔹 Atividade 1 – Cálculo de Bônus dos Funcionários

Programa que percorre uma lista contendo o valor das vendas de vários funcionários.

Para cada venda é verificado se a meta foi atingida.

Caso positivo, é calculado um bônus de **5%** sobre o valor vendido.

**Conceitos praticados:**

* Estrutura `for`
* Listas
* Estruturas condicionais
* Operações matemáticas
* Regras de negócio

---

### 🔹 Atividade 2 – Contagem de Vendas Acima da Meta

Programa responsável por identificar quantos dias ultrapassaram a meta de vendas estabelecida.

Foi utilizado um contador para registrar a quantidade de ocorrências.

**Conceitos praticados:**

* Contadores
* Estrutura `for`
* Condições (`if`)
* Processamento de listas

---

### 🔹 Atividade 3 – Comissão dos Vendedores

Programa que calcula automaticamente a comissão de cada vendedor.

Quando a venda ultrapassa a meta definida, é concedida uma comissão correspondente a **8%** do valor da venda.

**Conceitos praticados:**

* Listas
* Operações matemáticas
* Estruturas condicionais
* Estrutura `for`

---

### 🔹 Atividade 4 – Quantidade de Funcionários que Atingiram a Meta

Programa que percorre uma lista de vendas e contabiliza quantos vendedores atingiram a meta estabelecida.

**Conceitos praticados:**

* Contadores
* Estrutura `for`
* Condições
* Processamento de dados

---

### 🔹 Atividade 5 – Comissão com Bônus Extra

Programa que calcula a comissão dos vendedores utilizando duas metas.

Regras implementadas:

* Meta principal → comissão de **10%**
* Meta superior → comissão de **10%** + bônus fixo adicional

Este exercício introduz o conceito de **condicionais aninhadas** (`if` dentro de outro `if`).

**Conceitos praticados:**

* Condicionais aninhadas
* Estrutura `for`
* Regras de negócio
* Cálculo de bônus

---

### 🔹 Atividade 6 – Identificação da Maior Venda

Programa que percorre uma lista de vendas para identificar o maior valor registrado.

A solução foi desenvolvida utilizando comparação entre elementos durante a iteração da lista.

**Conceitos praticados:**

* Percurso de listas
* Comparação de valores
* Estrutura `for`
* Algoritmos de busca
